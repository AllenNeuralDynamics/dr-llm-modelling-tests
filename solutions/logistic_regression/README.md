# Logistic Regression

## Model
Logistic regression via scikit-learn's `LogisticRegression`, with second-degree
interaction terms generated by `PolynomialFeatures(degree=2, interaction_only=True)`.

## Rationale
Logistic regression is the gold-standard baseline for binary classification with
interpretable outputs. Coefficients map directly to log-odds, and exponentiated
coefficients give odds ratios that are intuitive to report. Adding pairwise
interaction terms (e.g., stimulus type x previous reward) lets the model capture
context-dependent effects without sacrificing interpretability -- each interaction
still has a single, signed coefficient.

## Features
Six boolean predictors: `previous_response`, `previous_reward`, `is_vis_target`,
`is_vis_non_target`, `is_aud_target`, `is_aud_non_target`.

With `interaction_only=True` at degree 2, the model also fits all 15 pairwise
products of these features (e.g., `previous_reward x is_vis_target`), for a total
of 21 features plus an intercept.

## Usage
```bash
uv run python logistic_regression/fit.py
```

Results are written to `logistic_regression/REPORT.md`.
